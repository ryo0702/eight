Button

Eight offers various Container Classes.
Using multiple responsive containers, you can lay the foundation for a high degree of freedom layout.

Default button

Container classes with different widths such as container, container-fluid, container-narrow are prepared.

Button size

Multiple sizes are available.Combined with .btn-block, you can express more sizes.

Button color

olor is set based on five colors of primary, info, success, warning, danger.use a color scheme similar to Bootstrap.

Other button

many kinds of buttons, such as Flat 3d style buttons.It is possible to specify it together with color and size, for example.

Detail

.btn Required Set up Button
.btn-xsmall Optional Set the size to xsmall
.btn-small Optional Set the size to small
.btn-middle Optional Set the size to middle
.btn-large Optional Set the size to large
.btn-xlarge Optional Set the size to xlarge
.btn-block Optional Display in full width
.btn-primary,etc Optional Set up color
.btn-noround Optional corners of the button are not rounded
.btn-skeuomorphic Optional skeuomorphic style design
.btn-flat3d Optional flat and 3d style design
